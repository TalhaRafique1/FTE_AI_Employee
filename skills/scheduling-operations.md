# Scheduling Operations Agent Skill (Silver Tier)

This skill enables the AI Employee to handle time-based operations and calendar management.

## Overview

The Scheduling Operations skill provides:
- Create and manage calendar events
- Set reminders and alerts
- Schedule recurring tasks
- Daily/weekly briefing generation
- Cron/Task Scheduler integration

## Integration with Orchestrator

```python
# In orchestrator.py

class SchedulingManager:
    """Manage scheduled tasks and reminders."""
    
    def __init__(self, vault_path: Path):
        self.vault_path = vault_path
        self.calendar = vault_path / 'Calendar'
        self.briefings = vault_path / 'Briefings'
        
    def check_scheduled_tasks(self):
        """Check and execute scheduled tasks."""
        schedules_folder = self.calendar / 'schedules'
        
        for schedule_file in schedules_folder.glob('*.md'):
            content = schedule_file.read_text()
            schedule_data = self._parse_schedule(content)
            
            if self._should_run(schedule_data):
                self._execute_scheduled_task(schedule_data)
    
    def generate_daily_briefing(self):
        """Generate daily briefing at scheduled time."""
        today = datetime.now().strftime('%Y-%m-%d')
        briefing_path = self.briefings / f'{today}_Daily_Briefing.md'
        
        # Gather data
        emails = self._count_emails()
        tasks = self._get_pending_tasks()
        events = self._get_todays_events()
        
        # Generate briefing
        content = self._generate_briefing_content(
            today, emails, tasks, events
        )
        
        briefing_path.write_text(content)
        return briefing_path
    
    def check_reminders(self):
        """Check and trigger due reminders."""
        reminders_folder = self.calendar / 'reminders'
        now = datetime.now()
        
        for reminder_file in reminders_folder.glob('*.md'):
            content = reminder_file.read_text()
            due = self._parse_due_date(content)
            
            if due and due <= now:
                status = self._get_status(content)
                if status == 'pending':
                    # Trigger reminder action
                    self._trigger_reminder(reminder_file)
```

## Reminder System

```python
# watchers/reminder_watcher.py

from base_watcher import BaseWatcher
from datetime import datetime
from pathlib import Path

class ReminderWatcher(BaseWatcher):
    """Watch for due reminders."""
    
    def __init__(self, vault_path: str, check_interval: int = 300):
        super().__init__(vault_path, check_interval)
        self.reminders_folder = self.vault_path / 'Calendar' / 'reminders'
        
    def check_for_updates(self) -> list:
        """Check for due reminders."""
        due_reminders = []
        now = datetime.now()
        
        for reminder_file in self.reminders_folder.glob('*.md'):
            content = reminder_file.read_text()
            due = self._parse_due_date(content)
            status = self._get_status(content)
            
            if due and due <= now and status == 'pending':
                due_reminders.append(reminder_file)
        
        return due_reminders
    
    def create_action_file(self, reminder_file: Path) -> Path:
        """Create action file for due reminder."""
        content = reminder_file.read_text()
        title = self._extract_title(content)
        
        action_content = f'''---
type: reminder_triggered
source: {reminder_file}
triggered: {datetime.now().isoformat()}
status: pending
---

# Reminder: {title}

This reminder is now due. Please take action.

Original reminder:
{content}
'''
        
        action_path = self.needs_action / f'REMINDER_{reminder_file.stem}.md'
        action_path.write_text(action_content)
        return action_path
```

## Best Practices

1. **Set realistic schedules** - Don't over-schedule tasks
2. **Time zone aware** - Always specify timezone
3. **Buffer time** - Add 15min buffers between meetings
4. **Review reminders daily** - Clear processed reminders
5. **Log all scheduled actions** - Maintain audit trail
6. **Test schedules** - Verify cron/tasks run correctly

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Scheduled task not running | Check scheduler service is running |
| Wrong timezone | Specify timezone in schedule file |
| Reminder not triggering | Check due date format |
| Briefing not generated | Verify vault path permissions |
